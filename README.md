# psu-asabe-robotics-2026
This repository contains all documentation, software, and design materials for the 2026 ASABE Student Robotics Competition, focused on the autonomous navigation and plant identification tasks in the simulated cornfield challenge.



The 2026 ASABE Competition:

"
Student teams will build autonomous robots that will race to create a stand count of a simulated corn field within 10-minute trials. Points will be earned for each plant type correctly identified and the correct action taken for each plant type. There will be a single plant, no plant/empty, and double plant. 

For the Standard Competition and Advanced Competition they will identify single, double, and empty plants. The Standard competition is required to remove the yellow plant when a double is detected. The Advanced Competition must remove the yellow plant when a double is detected and replant a seed when an empty plant  is detected. At the end of the run for both competitions the robot must display a stand count, representing how many plants they identified as single, double, or empty.
"
